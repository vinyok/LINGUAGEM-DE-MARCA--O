Servidor e Internet

Um sevidor de forma resumida, é apenas um computador com configurações específicas para servir.
Para acessar uma página da internet é precisa fazer uma requisição para o servidor, e para que eles possam
se comunicar é preciso falar a mesma linguagem, no caso HTML. Junto com o próprio navegador já vem
configurado para ler HTML.

HTML

HTML é a linguagem de marcação para hypertextos (Links).

Tags de HTML

Tags aprendidas na primeira aula:
Para quebrar linha: < br > - < /br >
Itálico: < i > - < /i >
Subinlhado: < u > - < /u >
Negrito: < b > - < /b >
